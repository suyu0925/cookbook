# 老豆腐

## 预处理

如果是先煎后烧的做法，切三角块最易入味。

## 烹饪

### 葱烧豆腐（隋坡版）

食材：
- 豆腐700克
- 葱白200克
- 油100克
- 大料2个
- 味精2克
- 酱油50克
- 水淀粉10克
- 葱油10克

预处理：
- 老豆腐切三角片，这个形状好入味。
- 葱白沿中间切开，斜切成6mm的眉毛葱。

开炒：
- 起锅烧油，不等油热，先下大料瓣，倒入一半葱白，开大火煸黄
- 葱煸黄后，关小火。豆腐顺着锅边倒入，把葱挤到旁边。
- 然后开中火，把豆腐煎至两面金黄
- 先倒入酱油，后加水，水位刚好没(mo4)过豆腐
- 加盐、味精，改小火，咕嘟3分钟
- 倒入剩余的葱白，勾芡后加入葱油，出锅

### 小院豆腐（老饭骨）

食材：
- 卤水老豆腐

配料：
- 姜片6-7片
- 大葱白斜刀切厚段，8-9段
- 八角8颗
- 酱油
- 黄酒
- 糖一平勺
- 白胡椒粉大概5抖
- 老抽
- 蚝油

烹饪：
- 豆腐用刀划一下，然后用手瓣成块
- 热锅冷油，先下姜片煸香
- 加一点猪油和鸡油，没有的话就算了
- 加入葱白段，煸香
- 下入八角，煸香
- 下酱油，把酱香味炒出来
- 下黄酒
- 加鸡汤，如果没鸡汤就清水加鸡精鸡粉
- 加一点蚝油
- 点一点老抽调颜色
- 加糖
- 加白胡椒粉
- 加入豆腐，盖盖，大火煮开，转小火（外圈小火）炖50分钟
- 将砂锅烧热，转到砂锅里上菜

### 砂锅炖豆腐（Fish吃了没）

食材：
- 砂锅豆腐最好，没有的话就用老豆腐自己煎一下，懒得煎的话买炸豆腐或家常豆腐也行
- 金针菇
- 黑木耳

配料：
- 干葱头最好，没有的话用洋葱
- 小米辣
- 香菜
- 大葱

预处理：
- 黑木耳和金针菇开水下锅，焯个十几秒水去生味
- 干葱头切两份，一份用来炸油，不用顾忌造型，随便切；另一份用来炖的，切成片
- 大葱切两份，一份用来炸油，斜刀切；另一份用来炖，只要葱白部分，切小段
- 香菜切两份，一份用来炸油，另一份用来炖，都切小段
- 小葱切两份，一份用来炸油，另一份用来炖，都切小段
- 小米辣竖着一劈二
- 豆腐切厚大片

炸油：
- 热锅冷油，油的量比正常炒菜多一点
- 把分出来炸油的那份干葱头、大葱、香菜、小葱下锅，全程大火，炒至金黄
- 加清水，热水冷水都行。小火让它煮着

烹饪：
- 砂锅烧热，给油，下五花肉，把五花肉炒至完全变色
- 下入大葱白，炒香
- 调一点酱油水（2克酱油，1克半水），等五花肉有点焦香的时候倒入上色，翻炒一下
- 下入砂锅豆腐，不要翻动
- 改最小火，把炸好的香料汤过滤进砂锅
- 把豆腐铺平，然后放入焯过水的木耳和金针菇
- 开大火煮沸
- 开始调味：盐3克、味精4克、白糖3克
- 盖盖，改中小火，煮3分钟煮到豆腐入味
- 开盖，加入另一份洋葱，以及小米辣
- 改大火冲一下，勾一个薄芡
- 出锅前再给一点香油
- 盖一个绿帽（另一份香菜和小葱），上菜


### 小白菜烧豆腐（隋坡版）

食材：
- 老豆腐
- 小白菜

配料：
- 水淀粉
- 酱油
- 盐
- 味精
- 大葱花
- 蒜米
- 花椒油

预处理：
- 豆腐切厚片（0.8厘米左右），多来点油，油温七成热，小火下入，然后开中火煎至两面金黄，盛出备用
- 小白菜掰成段，叶和帮分开放。起油锅少来点油，先下菜帮子，炒一会，撒点盐，让它出水。等菜帮子变色了，再下菜叶，炒出水后沥出备用。这一步是必须的，要去掉苦涩的菜水

烹饪：
- 油锅放入蒜米煸黄，再加大葱花，大葱花别炒久了，加入酱油，水（可没过豆腐的量），烧开
- 然后放入豆腐，加盐、味清，小火烧2分钟
- 豆腐烧软后，把小白菜盖在豆腐上，再烧1分钟
- 勾芡，加点花椒油出锅

### 沙茶酱焖豆腐（马小坏版）

食材：
- 豆腐，老嫩兼可

配料：
- 小葱花

碗汁：
- 1勺沙茶酱
- 2勺生抽
- 少许清水澥开

烹饪：
- 豆腐切成小方块，下锅煎至两面金黄
- 下碗汁，小火煮2分钟
- 下水淀粉，葱花，收下汁即可出锅
